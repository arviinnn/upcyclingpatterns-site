# Upcycling Patterns

**Upcycling Patterns** is an Erasmus+ KA210-SCH project focused on sustainability, environmental awareness, climate responsibility, and upcycling through international school collaboration.

## Project Information

- **Project Name:** Upcycling Patterns
- **Short Name:** UpCyc
- **Programme:** Erasmus+ KA210-SCH
- **Project Type:** School Education Small-scale Partnership
- **Duration:** 01/09/2025 - 31/08/2027
- **Website:** https://upcyclingpatterns.com/
