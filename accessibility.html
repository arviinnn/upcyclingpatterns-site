Accessibility Statement

Upcycling Patterns aims to make its Erasmus+ KA210-SCH project website accessible, understandable, and usable for as many visitors as possible.

1. Our Accessibility Approach

We aim to provide clear navigation, readable text, keyboard-friendly controls, responsive layouts, and accessible visual content. The site is designed with WCAG 2.1 Level AA guidelines in mind.

2. Features We Support

  • Responsive design for desktop, tablet, and mobile screens
  • Readable color contrast and dark mode support
  • Keyboard-accessible links, buttons, tabs, and form elements
  • Alternative text for important images
  • Reduced motion support for users who prefer less animation
  • Clear headings and structured page sections
  • Visible focus indicators on all interactive elements

3. Ongoing Improvements

Accessibility is an ongoing process. As the website grows with new project content, gallery items, downloads, and announcements, we will continue improving usability and accessibility.

4. Feedback

If you experience an accessibility issue on this website, please contact us and describe the problem so we can review it.
